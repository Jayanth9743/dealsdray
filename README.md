## Installation

Make sure you have the following software installed on your machine:

- [Node.js](https://nodejs.org/) (v14 or higher)
- [MongoDB](https://www.mongodb.com/) (or use MongoDB Atlas for cloud)
- [Git](https://git-scm.com/)

## frontend

- cd frontend
- npm install
- npm run dev

## backend

- cd backend
- npm install
- npm start
